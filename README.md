# Rustikub - A rummikub solver in Rust

[![Build Status](https://travis-ci.org/wemyss/rustikub.svg?branch=master)](https://travis-ci.org/wemyss/rustikub)

## Data file format
Colors:
```
b = black
l = blue
r = red
y = yellow
j = joker
```

Format:
```
blyr1 r9-11 y4-12 yrj13		# board
b1 r2 y2 y3 l9 l9 j			# hand
```
